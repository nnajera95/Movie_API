This API will provide users with information about different movies, directors, and genres. This will enable users to be able to create an account, update their info, as well as create a list of their favorite movies.

/movies this endpoint will return a list of all movies
/movies/[title] this endpoint will return a specific movie that has been searched
/movies/[genre]/[genreName] this endpoint will return a list of all the genres for movies
/movies/genre/[director] this endpoint will return a list of different directors from a specific genre
/register this endpoint will allow a new user to register for an account
/account/[information]/[update] this endpoint will return a users account information and allow them to update it
/account/[delete] this endpoint will give the user the ability to delete their account
/account/[favorites]/[add] this endpoint will allow a user to add a movie to their favorites list
/account/[favorites]/[remove] this endpoint will allow a user to remove a movie to their favorites list
