This API will provide users with information about different movies, directors, and genres. This will enable users to be able to create an account, update their info, as well as create a list of their favorite movies.

url/route param description method request response
/movies this endpoint will return a list of all movies get
/movies/[title] this endpoint will return a specific movie that has been searched get
/movies/[title]/[genre] this endpoint will return a genre with a list of movies in this category get
/director/bio this endpoint will return data about a director (bio, birth year, death) get
/register this endpoint will allow a new user to register for an account post
/account/[userId] this endpoint will return a users account information and allow them to update it put
/account/[delete] this endpoint will give the user the ability to delete their account put
/add/[favorite] this endpoint will allow a user to add a movie to their favorites list put
/remove/[favorite] this endpoint will allow a user to remove a movie to their favorites list delete
