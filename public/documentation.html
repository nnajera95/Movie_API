This API will provide users with information about different movies, directors, and genres. This will enable users to be able to create an account, update their info, as well as create a list of their favorite movies.

url/route param description method request response
/movies this endpoint will return a list of all movies get res.json(movieArray)
/movies/[title] this endpoint will return a specific movie that has been searched get {"title": "Happy Gilmore", "director": "Dennis Dugan", "genre": "Comedy/Romance"}
/genre/:Name this endpoint will return a genre with a list of movies in this category get {"Name": "Science Fiction", "Description": "Science fiction is a genre of speculative fiction that typically deals with imaginative and futuristic concepts such as advanced science and technology, space exploration, time travel, parallel universes, and extraterrestrial life."}
/directors/:Name this endpoint will return data about a director (bio, birth year, death) get {"Name": "Adam McKay", "Bio": "Adam McKay is an American film and television director, producer, screenwriter, and comedian.", "Birth": "1968/04/17"}
/users this endpoint will allow a new user to register for an account post {"newUser": "user information"} {"user has been added"}
/users this endpoint will return a list of all users get {User: "Targetmom3"}
/users/Username this endpoint will return a users account information and allow them to update it put {"UserId": "new userName"} "updated User"
/users/:Username/movies/:MovieID this endpoint will allow a user to add a movie to their favorites list post {"favoriteList": "movie to add"} res.send('movie added to favorites')
/users/:Username/movies/:MovieID this endpoint will allow a user to remove a movie to their favorites list delete res.send('movie removed from favorites')
/users/:Username this endpoint will give the user the ability to delete their account delete res.send('account deleted')
